# RISCV-Code
Repository for C/C++ and RISC-V Assembler code.

Code is developed and tested offline using Visual Studio Code editor with GitHub Copiot, C/C++, RISC-V Support, Mardown All in One, and other Extensions. GCC and riscv64-unknown-elf-gcc are primary compilers: qemu-risvc64 qemu-system-riscv64 and spike are primary simulators.
